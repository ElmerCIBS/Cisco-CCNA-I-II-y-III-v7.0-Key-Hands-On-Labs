# Cisco-CCNA-I-II-y-III-v7.0-Key-Hands-On-Labs
A quick reference of the core hands-on configuration labs completed throughout the CCNA curriculum using Cisco Packet Tracer and physical hardware
🛠️ CCNA 1: Introduction to Networks (ITN)
Initial Device Configuration: Securing Cisco routers and switches (hostnames, passwords, SSH, and banner messages).

IPv4 Subnetting: Designing, calculating, and implementing IP addressing schemes based on host requirements.

VLSM Design: Optimizing IP allocation to prevent address waste across LANs and WAN links.

IPv6 Addressing: Assigning Global Unicast (GUA) and Link-Local (LLA) addresses and verifying connectivity.

⚙️ CCNA 2: Switching, Routing, and Wireless Essentials (SRWE)
VLANs & Trunking (802.1Q): Segmenting broadcast domains and configuring trunk links between switches.

Inter-VLAN Routing (Router-on-a-Stick): Configuring router subinterfaces to enable communication between different VLANs.

EtherChannel (LACP/PAgP): Bundling physical links into a single logical link for redundancy and increased bandwidth.

DHCPv4 Server: Configuring a Cisco router to dynamically lease IPv4 addresses to network clients.

Static Routing (IPv4/IPv6): Implementing standard, default, summary, and floating static routes.

🔒 CCNA 3: Enterprise Networking, Security, and Automation (ENSA)
Single-Area OSPFv2: Implementing dynamic routing, configuring Router IDs, and tuning network parameters.

Access Control Lists (ACLs): Creating and applying Standard and Extended IPv4 ACLs for traffic filtering and security.

NAT for IPv4: Configuring Static NAT, Dynamic NAT, and Port Address Translation (PAT/Overload).

Device Hardening & VPNs: Securing management access and implementing basic site-to-site IPsec VPN tunnels.
