# Cisco-CCNA-I-II-y-III-v7.0-Key-Hands-On-Labs
A quick reference of the core hands-on configuration labs completed throughout the CCNA curriculum using Cisco Packet Tracer and physical hardware


This section outlines the primary hands-on configuration and troubleshooting labs completed throughout the three courses of the Cisco CCNA curriculum using Cisco Packet Tracer and real hardware.

---

### 🛜 CCNA 1: Introduction to Networks (ITN)
*   **Initial Device Setup:** Securing Cisco routers and switches (Hostnames, Passwords, Banner, and SSH access).
*   **IPv4 Subnetting & VLSM:** Designing and implementing efficient VLSM IP addressing schemes to prevent host waste.
*   **IPv6 Addressing:** Deploying Global Unicast (GUA) and Link-Local (LLA) addressing structures.

### 🔄 CCNA 2: Switching, Routing, and Wireless Essentials (SRWE)
*   **VLANs & Trunking:** Segmenting broadcast domains and configuring 802.1Q trunks between network switches.
*   **Inter-VLAN Routing:** Implementing Router-on-a-Stick (RoAS) topology via subinterfaces.
*   **EtherChannel (LACP):** Bundling physical links for increased bandwidth, high availability, and STP loop avoidance.
*   **DHCPv4 Server:** Provisioning routers to dynamically assign IP addresses, gateways, and DNS to network clients.
*   **Static Routing:** Building standard, default, summary, and floating static routes for IPv4/IPv6 path redundancy.

### 🔒 CCNA 3: Enterprise Networking, Security, and Automation (ENSA)
*   **Single-Area OSPFv2:** Configuring dynamic routing, altering Router IDs, and tuning hello/dead timers.
*   **IPv4 Access Control Lists (ACLs):** Creating Standard and Extended ACLs to secure traffic and restrict service access.
*   **NAT for IPv4:** Deploying Static NAT, Dynamic NAT, and Port Address Translation (PAT/Overload) for internet edge access.
*   **Device Hardening & VPNs:** Securing management access and establishing site-to-site IPsec VPN tunnels.
